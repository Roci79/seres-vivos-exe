¿Cómo estamos hechos los seres vivos?

Célula

Célula

Es el primer nivel en el que aparece la vida.
Es la unidad básica y estructural de los seres vivos.
Un organismo puede ser unicelular si está formado por una sola célula, o pluricelular si está formado por más de una.

Tejido

Tejido

Es un conjunto de células especializadas en una función concreta.
Algunos ejemplos de tejidos son:

  • Tejido epitelial
  • Tejido nervioso
  • Tejido conectivo
  • Tejido muscular

Órgano

Órgano

Es la unión de diferentes tejidos que trabajan juntos para realizar una función determinada.
Algunos ejemplos de órganos son:

  • El corazón
  • Los pulmones
  • El estómago

Sistema de órganos

Sistema de órganos

Es el conjunto de órganos que trabajan juntos para llevar a cabo una función concreta en el cuerpo.
Algunos ejemplos de sistemas de órganos son:

  • El sistema digestivo
  • El sistema respiratorio
  • El sistema nervioso

Organismo

Organismo

Es el conjunto de sistemas de órganos que forman un ser vivo completo.
Un organismo puede ser:

  • Una persona
  • Un animal
  • Una planta

Todos ellos están formados por células, tejidos, órganos y sistemas de órganos que trabajan en conjunto para mantener la vida.
